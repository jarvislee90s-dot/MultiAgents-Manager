# Getting Started

This guide covers the Getting Started feature of MultiAgents Manager.

## Overview

MultiAgents Manager is a unified management platform for AI programming tools, supporting Claude Code, Codex CLI, OpenCode, and OpenClaw.

## Installation

### macOS
1. Download the latest DMG file
2. Drag to Applications folder
3. Grant Accessibility permissions on first launch (for terminal focus)

### Linux
1. Download the AppImage file

3. Run: 

## Usage

After launching, the dashboard automatically scans installed AI programming tools and displays active sessions. You can view session status, click cards to jump to terminals, manage Skill/MCP/Plugin resources, and create preset groups for one-click switching.

## Troubleshooting

If sessions don't appear:
1. Ensure at least one AI tool is running
2. Check app has process access permissions
3. Verify tool detection is enabled in Settings
